## 常见问题 FAQ

**Q：搬瓦工官网打不开怎么办？**

A：搬瓦工主域名在大陆有 DNS 污染问题，有几个官方镜像站可以正常访问，效果和官网完全一样，购买和账户管理都在上面操作，不影响使用。

**Q：CN2 GT 和 CN2 GIA 区别到底有多大？**

A：CN2 GT 是电信国际出口的商业线路，会走普通国际骨干，晚高峰拥堵时容易掉速。CN2 GIA 是更高优先级的专线，有独立通道，晚高峰期基本不受普通流量拥堵影响。简单说：白天差别不大，晚高峰差别很明显。

**Q：搬瓦工的流量超了会怎样？**

A：套餐流量用完后，服务器会被限速，等到下个计费周期重置后恢复正常。不会额外扣费，也不会直接停机。限量版的一些套餐流量稍少，使用前要算清楚自己的需求。

**Q：类似搬瓦工的 VPS 有没有？**

A：纯从价格上看有便宜的，但线路质量达到 CN2 GIA 的在这个价位段真的少。市面上有几家做 CN2 GIA 的，要么库存长期缺货，要么价格比搬瓦工贵，要么需要配置更复杂的网络设置。如果你核心诉求是 CN2 GIA 线路 + 支持支付宝 + 随时有货，搬瓦工目前是最稳的选择。

**Q：搬瓦工支持哪些付款方式？**
